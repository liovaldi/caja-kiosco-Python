Sistema de Caja-Kiosco
Aplicacion desarrollada en Python que simula el funcionamiento de una caja de kiosco, permitiendo registrar ventas, aplicar descuentos y calcular totales mediante un menú interactivo en consola.

Funcionalidades:
Regsitro de productos en cantidades
Cálculo de total de compras
Aplicacion de descuentos
Resumen final de operacion

Logica implementada:
Estructuras condicionales
Bucles while 
Validacion de datos ingresados
Cálculo de totales y descuentos
Almacenamiento de datos en variables

Tecnologias:
Python - Visual Studio Code

Objetivo: 
Aplicar logica de programacion en la simulacion sencilla de un sistema de ventas real, incorporando manejo de datos y control de flujo.

Mejoras futuras:
Separacion en funciones
Uso de listas
Persistencias de datos
Interfaz mas avanzada
